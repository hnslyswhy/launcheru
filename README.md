## Tech Stack

**Client:**  React.Js, React Hooks, React Router, Axios, Sass, REST API 

**Server:**  NodeJS, Express, Nodemon, Passport, Passport-google-oauth20, Helmet, Morgan


## Installation

1. Clone this repo
2. Setup backend

```bash
  cd server
  npm i
  npm run watch
```
3. Setup frontend
```bash
  cd client
  npm i
  npm start
```
